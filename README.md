# mongo_db_learning
### run mongo service
1. go to docker dir
2. run
```commandline
docker-compose up
```
### run commands
1. run the service
2. run python/scripts command, eg.
```commandline
python/scripts count_documents.py --db mydb --collection tweets
```
### shut service down
1. go to docker dir
2. run
```commandline
docker-compose down -v
```
